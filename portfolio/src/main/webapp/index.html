Projects

Radar Material Classifier

Project done over the course of a semester under the guidance of my professor for Humanoids, a freeform class where we were encouraged to explore whatever interested us in the CS/Robotics field. Uses TensorFlow and a high frequency radar hosted on a Raspberry Pi in order to sense the densities and other characteristics of a material in order to classify them accordingly. Got to greater than 90 percent accuracy but needed more accurate equipment which unfortunately could not be acquired due to not being on campus because of coronavirus :(

QR Enabled ATM

Winning hackathon project created at Capital One's software engineering summit that takes a qr code generated from an android app with all the transaction info for a withdrawal or deposit and updates the users account balance automatically after tansaction completes. Uses a raspberry pi as the "ATM"
Github Link

LeapMotion Base Defence

Hackathon project done at Hack112, a class sponsored hackathon. Used a LeapMotion hand sensor in order to track hand rotation which was used in order to rotate a shield to block projectiles of consistently increasing difficulty. Utilizes threading in order to play sounds when hit. Players of the game called it extremely intuitive and fun.
Github Link

GPS Loop Detection

Final term project in which I created an application where someone could take a gpx recording of some outdoor activity such as mountain biking or hiking and the application would find all the differing loops in the trail and categorize them according to difficulty. This was all stored in a backend SQL database where multiple users could store their gps data seperately and could be selected upon login.
